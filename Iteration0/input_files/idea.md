**Title**: Residual Deep-Stack MLP with Hybrid Huber-Log Spectral Loss

**Description**: This architecture employs a high-capacity residual MLP with a 6→512→1024→1024→512→20996 topology, utilizing GELU activations and LayerNorm to capture the high-frequency acoustic oscillations across 6,000 $\ell$-modes. To address the sign-indefinite nature of $C_\ell^{TE}$ while maintaining precision for the strictly positive $C_\ell^{TT, EE, PP}$, the training objective implements a dual-regime loss function: MSE on log-transformed $D_\ell$ for TT, EE, and PP, combined with a Huber loss on linear $D_\ell^{TE}$ to prevent gradient instability at zero-crossings. By increasing the depth to four primary residual blocks, the model maximizes feature extraction capability within the provided Blackwell GPU memory constraints. The final inference pipeline is optimized via JIT-compilation to satisfy CPU latency requirements, mapping inputs directly to the 20,996-dimensional spectral space without bottleneck-induced information loss. This approach ensures likelihood-level accuracy by optimizing the spectral reconstruction as a high-fidelity regression task rather than a dimensionality reduction problem.
